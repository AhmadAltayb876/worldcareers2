الوصف الكامل

يقوم مبرمج Backend بتصميم وتطوير وصيانة الأنظمة الخلفية لتطبيقات الويب والمواقع الإلكترونية. يشمل ذلك إنشاء قواعد البيانات، تطوير واجهات برمجة التطبيقات (APIs)، وضمان تكامل الأنظمة مع الواجهات الأمامية. يعمل المبرمج على تحسين أداء الخوادم، تأمين البيانات، وضمان استقرار واستجابة النظام. يتعاون مع فرق التطوير الأخرى لضمان تحقيق الأهداف التقنية والتجارية للمشاريع.

المهارات المطلوبة

  • إتقان لغات البرمجة مثل Python، Java، أو Node.js
  • خبرة في إدارة قواعد البيانات مثل MySQL أو MongoDB
  • معرفة بأطر العمل مثل Django أو Spring Boot
  • فهم جيد لأمن المعلومات والبنية التحتية السحابية
